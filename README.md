# OC Team Workflow Research

Research và templates cho hệ thống quản lý team với AI agents.

## 📋 Flow Tổng Hợp

```
Step 0: Wi + Hina discuss project idea
    ↓
Step 1 (OPTIONAL): Gather Advisors/Experts
    ↓
Step 2: Define Project (Project Charter Template)
    ↓
Step 3: Identify Required Roles (Role Definition Template)
    ↓
Step 4: Build Team (Subagent Template)
    ↓
Step 5: Setup Team (Role Map + Team Workflow)
    ↓
Step 6: Onboard Members (Onboarding Checklist)
    ↓
Step 7: Start Execution
```

## 📁 Templates

| # | Template | File | Purpose |
|---|----------|------|---------|

| 2 | Project Charter | `01-project-charter.md` | Định nghĩa project |
| 3 | Role Definition | `02-role-definition.md` | Mô tả vai trò |
| 4 | Subagent Template | `03-subagent-template.md` | Template tạo AI agent |
| 5 | Role Map | `04-role-map.md` | Mapping roles → people/agents |
| 6 | Team Workflow | `05-team-workflow.md` | Quy trình làm việc |
| 7 | Onboarding Checklist | `06-onboarding-checklist.md` | Checklist onboard |
